## Getting Started

### Prerequisites

- Git
- A runtime appropriate for the stack listed above

### Installation

```bash
git clone https://github.com/RaghavSobti37/Resume-Generator.git
cd Resume-Generator
npm install
```

### Development

```bash
npm run dev
```

### Production Build

```bash
npm run build
```
